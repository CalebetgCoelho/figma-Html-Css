Wireframing
in Figma

This file showcases examples on how you could easily
build wireframes in Figma. For more tutorial content and
features overview, visit help.figma.com.

Overview

This file contains common modular layouts that could be used to assemble web page wireframes.
Choose from various navigation, hero, and body content modules.

Get started using this file

1. Create a new 1680px wide Frame which will serve as the container for your wireframe.

2. Duplicate a Frame you want to use from the assortment to the right. A sample wireframe is included.

3. Create custom modules using Components already included in this file by going to the Assets panel in the top-left.
